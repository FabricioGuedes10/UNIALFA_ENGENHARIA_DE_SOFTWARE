# NotaLembrete

Projeto da disciplina **Fundamentos de Engenharia de Software** - UNIALFA

## Descrição
Protótipo de um aplicativo de anotações e lembretes desenvolvido para a disciplina. O objetivo é demonstrar o fluxo de telas e funcionalidades principais.

## Funcionalidades Planejadas
- Criar, editar e excluir notas
- Definir lembretes com data e hora
- Interface simples e intuitiva

## Mockups
As telas do aplicativo estão na pasta `/FUNDAMENTOS_DE_ENGENHARIA_DE_SOFTWARE/NOTALEMBRETE/IMAGENS`

## Status do Projeto
Este repositório contém apenas os protótipos de tela. Não há código implementado no momento.
